# Jogo

Instruções de Compilação e execução
Para compilar o projeto, certifique-se de que a biblioteca Raylib está configurada, clone nosso repositório disponível em: Github e execute o comando abaixo no terminal    gcc src/*.c -o pequemen.exe -O1 -Wall -std=c99 -Wno-missing-braces -I    C:/raylib/raylib/src -L C:/raylib/raylib/lib -lraylib -lopengl32 -lgdi32 -lwinmm
Após isso, execute o programa nomeado “pequemen.exe” e se divirta :)
